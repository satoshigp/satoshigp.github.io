ダメージの計算

-今まで-

敵に自身(プレイヤー)の攻撃が当たった際、 定数値でダメージの計算(ほぼ計算してない・・・)を行っていた。
※例のスクリプト
OnTriggerEnter(other) {
//相手のタグ判定
if(other.comparetag("teki")) {
//ダメージ計算？
HP-=dmage;
}
}
結論....めちゃ不便！
-メリット-
・簡単
-デメリット-
・プレイヤーも攻撃力や防御力等があった際計算ができなくなってしまう
・ ステータス更新時にとても不便！
etc...(思いうカバン！)
