CSSをのanimationやtransitionを使用せずに動的な動きを作ってみよう！
JavaScriptとは？
  • Webブラウザ上で起動するプログラミング言語であり オブジェクト指向プログラミング言語です。
  • 最近ではWebブラウザ上だけではなく、サーバーサイド側でも動くNode.js
  • IoTのデバイスコントロール言語として話題です！
  • JavaScriptは インタープリタ言語です。
    • ※インタープリンタ言語の他にもコンパイラ言語も存在します。
    名前用途言語
    コンパイラ言語プログラムをコンピュータが理解できる形式に変換するプログラム言語C++,C#,Java
    インタープリンタ言語プログラムをコンピュータが理解できる形式に逐次翻訳しながら、そのプログラムを実行してゆくプログラム言語(スクリプト言語とも言う)JavaScript,Ruby,PHP

    簡単に説明すると

    コンパイラ言語:一回一回ビルドする。
    インタープリンタ言語：コードを書いたら即反映！ビルドしない！
    スライドショーを実行する前の準備
    CSS側でやるべきこと
    スライドショーのフレームを作成します。
    参考スクリプト）
    CSS 
                //見える範囲の設定
            .slider-frame
            {
              内側、外側の余白の設定(なくてもOK！)
              padding:0px;
              margin:0px;
              //外側の範囲の設定
              width:500px;
              height:500px;
              //はみ出たところは消す！
              overflow:hidden;
            }
            //動く部分の設定
            #slider-wall
            {
              //中身の範囲設定
              width:300%;
              height:500px;
              //変化の度合い設定
              transition:ease 3s;
            }
            //画像一つ一つに設定
            .slider-item
            {
              //画像の範囲設定
              width:500px;
              height:500px;
              //左ならべに設定
              float:left;
            }
    上記のスクリプトをCSSで設定します 実際に実行してみるとこんな感じ！
    まだ動的な処理はしていないので動きまん。
