【サイト上でアニメーションを作成してみよう】

-前提条件-

アニメーションには2パターンの方法があります！
  • Transitionについて
  • Animationについて

    • Animation(アニメーション)とは？
    Animationとは？ ほぼほぼtransitionと動きは変わらない気がしますが
    イベントハンドラを用意しなくとも動的処理が可能です。　
    しかし@keyframeを設定してあげないと動作はしません。
    任意でアニメーションを止めることはできません。

    animationのプロパティ

    プロパティ名読み効果
    name ネームキーフレームの名前を指定します
    delay ディレイ変化が開始されるまでの時間
    duration デュレイション 変化が始まって終わるまでの時間を指定します。
    timing-function タイミングファンクション変化の進行度
    iteration-countイテレーションカウントアニメーションの再生回数を設定します
    direction ダイレクションアニメーションを交互に反転再生させるかどうかを指定する
    例）画像のスライド
    animation-name
               要素名A,クラス名
           {
              animation-name:slide
           }

           @keyframe slide
           {
             処理だーーーーーーー
           }
    初期値はnone。
    @keyframeで作成した関数名を指定してあげればOK！
    animation-delay
                     要素名a,クラス名
                 {
                     animation-delay:値（数値）
                 }
    値は以下の表で設定することができます。
    指定できる単位単位/ｓ
    s1sは1秒
    ms 1msは1000分の1秒
    animation-duration
                             要素名a,クラス名
                         {
                             animation-delay:値（数値）
                         }
    値は以下の表で設定することができます。
    指定できる単位単位/ｓ
    s1sは1秒
    ms 1msは1000分の1秒
    animation-timing-function
                                 要素名A,クラス名
                             {
                                 animation-timing-function:値
                             }
    変化の進行度はベジェ曲線で表すことができます。
    ease 変化の開始と終了付近の動きを滑らかに。
    linear 一定で変化します
    ease-in 変化開始付近の動きを緩やかにします
    ease-out 変化終了付近の動きを緩やかにします
    ease-in-out 変化の開始と終了付近の動きを緩やかにします
    cubic-bezier()任意で変化量を決めることができます。
    animation-iteration-count
                     要素名A,クラス名
                 {
                   animation-iteration-count:値;
                 }
    値効果
    infinite 無限にアニメーションを繰り返します
    指定回数 指定回数分アニメーションを繰り返します
    animation-direction
                     要素名A,クラス名
                 {
                   animation-direction:値;
                 }
    値は以下の表で設定できます！
    値 効果
    normal通常の方向で再生されます
    revarse逆の方向で再生されます
    altenateアニメーションの繰り返し回数が奇数の場合通常方向、
    偶数の場合は逆方向に再生されます
    altenate-revarseアニメーションの繰り返し回数が偶数の場合通常方向、
    奇数の場合は逆方向に再生されます
