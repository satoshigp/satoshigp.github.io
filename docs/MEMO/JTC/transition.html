    • transition(トランジション)とは？
    transitionとは？

    transitionとは直訳すると「変化」,「変わる」ということです。
    プロパティーを一つ一つ決めることもできますし、
    まとめてプロパティを決めることも可能です。
    イベントハンドラ(フラグ)を設定しないとほぼ使えないです。
    要素名A,クラス名{transition:-property-delay-duration-timing-function}
    プロパティ名 読み 効果
    property プロパティ transitionを適応するプロパティの指定
    delay ディレイ変化が開始されるまでの時間
    duration デュレイション 変化が始まって終わるまでの時間を指定します。
    timing-function タイミングファンクション変化の進行度
    例）画像にマウスを乗せたら画像が大きくる等 こういった動的な要素を簡単に作成出来るのがtransitionの特徴です
    01.transition-property 
    要素名A,クラス名
     {
       transition-property:適応させたいプロパティ
     }
    transition.propertyのプロパティーは以下のようになります
    all transitionが適応可能なプロパティすべてに効果を適応。
    none どのプロパティにも適応させない
    transition-delay
              要素名a,クラス名
          {
              transition-delay:値（数値）
          }
    値は以下の表で設定することができます。
    指定できる単位単位/ｓ
    s1sは1秒
    ms 1msは1000分の1秒
    transition-duration
                      要素名a,クラス名
                  {
                      transition-delay:値（数値）
                  }
    値は以下の表で設定することができます。
    指定できる単位単位/ｓ
    s1sは1秒
    ms 1msは1000分の1秒
    transition-timinf-function
                          要素名A,クラス名
                      {
                          transition-timinf-function:値
                      }
    変化の進行度はベジェ曲線で表すことができます。
    ease 変化の開始と終了付近の動きを滑らかに。
    linear 一定で変化します
    ease-in 変化開始付近の動きを緩やかにします
    ease-out 変化終了付近の動きを緩やかにします
    ease-in-out 変化の開始と終了付近の動きを緩やかにします
    cubic-bezier()任意で変化量を決めることができます。
